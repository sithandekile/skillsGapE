# Skill Gap Equalizer Platform

A MERN stack platform where employers post real-world problems and job seekers showcase their skills through solutions, building reputation based on performance rather than resumes or age.

##  Features

### For Employers
- Post real-world problems with detailed requirements
- Set rewards (jobs, internships, contracts, cash)
- Review submissions and provide feedback
- Award contracts/jobs to the best solutions
- Track problem engagement and submissions

### For Job Seekers
- Browse available problems by skills and difficulty
- Submit solutions with code repositories and demos(future feature)
- Work individually or form teams
- Build reputation through successful solutions
- Get discovered by employers based on actual performance

### Platform Features
- JWT-based authentication
- File upload system using GridFS
- Real-time problem and submission tracking
- Reputation scoring system
- Skill-based matching and recommendations
- Responsive design with Tailwind CSS

## 🛠️ Tech Stack

**Backend:**
- Node.js & Express.js
- MongoDB with Mongoose
- JWT Authentication
- Multer for file uploads
- Bcrypt for password hashing

**Frontend:**
- React with Hooks
- Tailwind CSS for styling
- React Icons for icons
- React Router for navigation
- Axios for API requests
